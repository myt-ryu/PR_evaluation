# 科学技術コミュニケーションにおける効果指標と評価手法の現状・課題、および解決策の包括的分析

科学技術コミュニケーション（以下、SC）は、2005年以降、日本において「科学の公衆理解（PUS）」から「双方向の対話」へとその軸足を移してきた。しかし、活動の普及・発展に伴い、個々の実践の質を客観的に評価し、改善に結びつけるための包括的な評価枠組みの不備が顕著な課題となっている。本資料では、提供されたソースに基づき、SCにおける評価の現状、課題、および有効な解決策としてのセグメンテーション手法や包括的枠組みについて詳述する。

---

## 1. 科学技術コミュニケーション評価の現状と課題

SCの実践は、サイエンスカフェから政策対話まで極めて多様であり、一貫した評価手法体系の構築を困難にしている。

### 1.1 現状における評価の不備

SCの評価は、現状では個別の実践者の試行錯誤に委ねられており、以下のような場当たり的かつ一過性の評価に留まっている。

- **限定的な指標**: アンケートによる「満足度」の5段階評定などが主だが、満足度をもたらす具体的要因や改善指針が不明確である。
- **多層的な視点の欠如**: 一般参加者だけでなく、資金提供者（リソース提供主体）、協力研究者、実施組織といった多様なステークホルダーのニーズを反映できていない。

### 1.2 評価の整備を阻む要因

評価手法の確立が困難な背景には、SC特有の以下の要因が存在する。

| 要因                     | 詳細内容                                                                                                 |
| :----------------------- | :------------------------------------------------------------------------------------------------------- |
| **目的の多様性**         | 科学教育、広報、参加型テクノロジーアセスメントなど、実践ごとに目的の配合比率が異なる。                   |
| **双方向性のジレンマ**   | 双方向性を重視するほど、目標設定が参加者との相互作用でダイナミックに変化するため、事前の指標設定が困難。 |
| **アクターの多様性**     | 実践者（研究者、広報、市民団体等）ごとに評価の観点が異なり、共通言語が存在しない。                       |
| **トランス・サイエンス** | 科学だけでは解決できない複雑な社会問題を扱うため、評価指標の選定自体が高度な判断を要する。               |

### 1.3 評価の不備がもたらす問題

- **比較可能性の欠如**: 異なる実践を相互比較する基盤がないため、ノウハウの共有や実践全体の発展が阻害される。
- **説明責任（アカウンタビリティ）の不足**: 公的資金や寄付金に対する具体的な投資対効果を示すことが難しい。
- **キャリアの不安定さ**: 実践者が正当な評価を得られないことが、コミュニケーターの地位やキャリアを不安定にしている。

## 2. セグメンテーション手法（VSEG）による解決案

実践者が現場で容易に利用でき、かつ参加者層を客観的に評価する手法として、オーストラリアのヴィクトリア州政府が開発したセグメンテーション（VSEG）の有効性が示されている。

### 2.1 VSEGの構成

VSEGは、わずか3つの質問項目から市民を6つのセグメントに分類する。

| 質問内容                      | 回答による分類の鍵             |
| :---------------------------- | :----------------------------- |
| 1. 科学・技術に関心があるか   | 関心の有無と程度。             |
| 2. 情報を積極的に調べるか     | 能動的な関与の有無。           |
| 3. 情報を見つけ、理解できるか | 情報アクセス能力と自己効力感。 |

### 2.2 VSEG活用の利点と知見

- **実践的容易性**: 質問数が少ないため、イベント現場での参加者負担が極めて低い（Kawamotoらの10問モデル等と比較して優位）。
- **参加者層の可視化**: 普段科学に関与しない層（低関与セグメント）をどれだけ引き込めているかを定量的に把握可能。
- **時間的安定性**: 高関与層および低関与層については、時間経過によるセグメント変化が少なく、評価の信頼性が高い。
- **相関指標**: 高関与セグメントほど、専門用語への関心が高く、イベント参加への内発的動機や自己効力感（内容理解や積極的関与への自信）を高く予期する傾向がある。

## 3. 包括的評価枠組みの構築とロジックモデル

SCの評価を体系化するためには、評価を「情報を意思決定主体が利用可能な形に変換する行為」と再定義し、包括的な枠組みを適用する必要がある。

### 3.1 評価プロセスの構造化

評価を以下の要素に分解し、それぞれの階層で適切性を検証する。

1. **評価対象の同定**: 実践、成果物、実践主体、参加主体のいずれを対象とするか。
2. **意思決定主体の特定**: 誰がその評価結果を使って、どのような資源（予算・人員・時間）の配分を決定するのか。
3. **変換プロセスの設計**: 生のデータをどのように「利用可能な情報」に加工するか。
4. **介入の可能性**: 評価結果に基づき、次回の実践を改善（中断・継続・変更）できる余地があるか。

### 3.2 事後的なプログラム理論（ロジックモデル）の描出

SC実践、特に双方向対話を重視する活動においては、当初の計画通りの評価（総括的評価）だけでは不十分である。

- **事後的再解釈**: 実践プロセスで生じたダイナミックな変化を、終了後にロジックモデル（入力→活動→出力→アウトカムの連鎖）として描き直す。
- **設計妥当性の検証**: 描出されたロジックモデルに基づき、当初の目的と実施内容、得られた成果の整合性を事後的に検証し、次回の設計へフィードバックする。

## 4. PR効果測定アプリケーション開発への示唆

本研究の成果は、SCや広報におけるPR効果測定アプリの開発において、以下の機能を実装する際の理論的基盤となる。

### 4.1 推奨される機能要素

- **超軽量アンケート機能**: VSEGの3問をベースに、参加者の属性（関与度レベル）を瞬時に特定する機能。
- **多層的なダッシュボード**:
    - 「参加者向け（満足度・自己効力感）」
    - 「実践者向け（ノウハウ共有・目標達成度）」
    - 「リソース提供者向け（アカウンタビリティ・投資対効果）」
    の3視点でデータを切り替え表示する。
- **動的ロジックモデル・エディタ**: 実践の進行状況に合わせて、目的や予想アウトカムを修正・記録し、事後的に一貫性を分析できる機能。
- **キーワード関心度分析**: 専門性の高い用語（Life Innovation等）と、日常生活に密接した用語（Relevance）への関心を比較し、コンテンツの「刺さり具合」を測定する。

### 4.2 評価情報の「変換」アルゴリズム

単なる集計値ではなく、意思決定（例：次回の開催規模を拡大すべきか、ターゲット層を変更すべきか）をサポートするための、評価基準に基づく「レコメンド機能」の搭載が望まれる。

> **重要引用**:  
> 「評価とは、ある対象の情報を、その対象に影響を与える意思決定において、意思決定主体が利用可能な情報に変換する行為である。」（石村 2018）  
> 「セグメント間には……科学・技術イベントへの参加意向、動機づけなどにおいて、……想定されるような違いがみられる。」（後藤ら 2014）
